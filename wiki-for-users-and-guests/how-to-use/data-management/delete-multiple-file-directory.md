# Delete Multiple File, Directory

![](<../../../.gitbook/assets/image (23).png>)

삭제 할 디렉토리와 파일을 선택하고 우측의 삭제 버튼을 클릭합니다.



![](<../../../.gitbook/assets/image (24).png>)

삭제 대상 파일/디렉토리 목록을 확인하고 삭제 버튼을 클릭합니다.



![](<../../../.gitbook/assets/image (25).png>)

파일 및 디렉토리가 삭제되었습니다.
