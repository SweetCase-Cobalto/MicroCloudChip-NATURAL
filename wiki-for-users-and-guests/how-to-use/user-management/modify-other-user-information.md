# Modify Other User Information

{% hint style="warning" %}
해당 기능은 관리자만 사용할 수 있습니다.
{% endhint %}

![](<../../../.gitbook/assets/image (5).png>)

Account 탭에서 수정하고 싶은 계정을 선택합니다.

우측에 있는 수정 버튼을 클릭합니다.



![](<../../../.gitbook/assets/image (6).png>)

계정 수정 창이 나타나면 클라이언트 계정을 수정할 수 있습니다.
